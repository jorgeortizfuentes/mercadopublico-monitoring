Acerca de Mercado Público Monitor

Descripción

Mercado Público Monitor es una herramienta diseñada para facilitar el seguimiento de licitaciones públicas en Chile. Permite la búsqueda automatizada y el análisis de oportunidades de negocio en el mercado público.

Características

  • Búsqueda automatizada de licitaciones
  • Filtrado por palabras clave
  • Análisis estadístico
  • Interfaz intuitiva
  • Exportación de datos

Tecnologías

  • Backend: Python, FastAPI, SQLAlchemy
  • Frontend: HTML5, Bootstrap 5, JavaScript
  • Base de datos: SQLite
